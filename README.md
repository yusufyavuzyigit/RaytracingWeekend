# cpp-template
A C++ project template using CMake and CPM. It includes:
- `Core` library target
- `App` executable linking `Core`
- `Tests` using Catch2 via CTest
- `Benchmarks` using Google Benchmark

## Prerequisites
- CMake 3.60+ (recommended)
- A C++20-compatible compiler
- Git (for dependency fetching via CPM)
- Optional tools
	- Ninja (faster builds)
